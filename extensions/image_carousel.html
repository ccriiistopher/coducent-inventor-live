Image Carousel

Extension price: 5
It is supported for:KodularAppInventorNiotron

This extension provides a widget that allows you to create a horizontal carousel of images. It has several features, such as automatic playback (switch to next image). In addition, it includes several interface features that can be edited.

Some features:

  • Animation of scaling when changing images.
  • Autoplay.
  • Image scaling (Centered and Adjusted).
  • Endless mode, the same images will be repeated over and over again.
  • Image Caching
