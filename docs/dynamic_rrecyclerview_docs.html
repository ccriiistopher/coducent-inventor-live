CreateGridManager

Create and return a new GridLayoutManager instance, which will allow us to display the RecyclerView as a Grid layout

spanCount: Number of columns (when it is vertical) or rows (horizontal)

stackFromEnd will specify whether the RecyclerView is going to take the bottom side as starting.

horizontalBoolean-
reverseBoolean-

*LayoutManager-

CreateStaggeredGridManager

Create and return a new StaggeredGridLayoutManager instance, which will allow us to display the RecyclerView as a Staggered Grid layout

spanCount: Number of columns (when it is vertical) or rows (horizontal)

Unlike the GridLayoutManager, this manager does not necessarily need the items to have the same size, since it can display them depending on their content.

NameType(s)Def. value
spanCountNumber-
horizontalBoolean-
reverseBoolean-
