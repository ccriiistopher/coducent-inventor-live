Image Carousel Extension Documentation

Events

OnItemChanged

Event fired every time the focused item on the carousel changes.

NameType(s)Def. value
positionnumber-

ItemClicked

Event fired every time an Item is clicked.

NameType(s)Def. value
positionnumber-

Methods

Build

Adds the Image Carousel to the screen and initializes it with the indicated list.

NameType(s)Def. value
componentcomponent-
imageslist-

NotifyDataUpdate

Notifies Carousel that the list it was initialized with has changed (items were added or removed).

If the list is a global variable, it is preferable to always work on that same list, that is, modify it by adding or removing items, but not by replacing the value of the variable with a new list.

Stop

For an Image Carousel with the 'AutoPlay' option, this block pauses the automatic switching of images.

Play

For an Image Carousel with the 'AutoPlay' option, this block resumes the automatic switching of images.

IsPlaying

For an Image Carousel with the 'AutoPlay' option, this block returns true or false depending on the state of the autoplay.

Useful for working with 'Play' and 'Stop' blocks

Next

This block changes the focused item to the next one, by smooth scrolling to the next position.

Previous

This block changes the focused item to the previous one, by smooth scrolling to the previous position.

GoToPosition

This block changes the focused item to the item at the indicated position, if the smooth option is True then it will perform a smooth (animated) scroll to the position, if False then it will quickly change without any animation.

NameType(s)Def. value
positionnumber-
smoothBoolean-

ImageWidthByCount

This block calculates a width for each image, depending on the size of the container (where the Carousel is located) and the number of items you want to display.

Offset: It is an extra size that will be added to the calculated result. Leave it at 0 if it is to be ignored.

NameType(s)Def. value
containerComponent-
visibleItemCountnumber-
offsetnumber-

ImageWidthByPercent

This block calculates a width for each image, depending on the size of the container (where the Carousel is located) and a percentage (relative to the container.)

NameType(s)Def. value
containerComponent-
percentnumber-

NameType(s)Def. value

Sets the margin of each carousel item on the corresponding side

NameType(s)Def. value

Offset

Sets an additional offset space to display at the start and end of the carousel.

Leave at 0 if no space is required to be displayed

Default value: 250

NameType(s)Def. value
offsetnumber-

ImageScaling

Sets the scaling ratio of the image (the picture file).

It is quite useful to prevent performance problems with very large images.

Values ​​range from 0... to 1

Default value: 0.5

NameType(s)Def. value
scalingNumber0.5

AnimationScale

In carousels with the ChangeAnimation property activated, with the help of this property it is possible to specify to what size items that are not in focus are to be scaled.

Values ​​range from 0... to 1 where 1 is the actual size without scaling.

Default value: 0.85

NameType(s)Def. value
scalingnumber0.85

ImageBackgroundColor

This property sets the background color of each item.

NameType(s)Def. value
colorColor#FFFFFF

Endless

This property sets whether the carousel will end depending on the list of data. If it is activated, then the items will be repeated over and over again cyclically.

Default value: False

NameType(s)Def. value
endlessBooleanFalse

CenterCrop

This property is focused on the mechanism to load the images in each item.

If enabled, the image will be centered and cropped to fit each item perfectly.

If disabled, the image will fit the size of the item without cropping it.

Default value: False

ChangeAnimation

If enabled, the carousel will produce an animation each time a scroll occurs or the focused item changes. The animation consists of scaling the unfocused items to a smaller size (depending on the value of 'AnimationScale') and scale the focused item to its actual size.

Default value: False

LinearSnap

By default, the carousel has a pager-like scrolling behavior. With this property, we can change that behavior.

If on, switches to snap swipe behavior.

Default value: False

AutoPlay

This property specifies whether the carousel changes to the next item automatically.

The interval is set with the 'PlayDelay' property

Default value: False

PlayDelay

This property specifies the time interval in which an automatic change in the carousel will occur.

Its value is in milliseconds.

Default value: 2000 (2s.)
