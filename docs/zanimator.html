ZAnimator

It is supported for:KodularAppInventorNiotron
It is part of Fancy RecyclerView Widgets
To get this extension, please go to: Fancy RecyclerView Widgets

Posted on January 25, 2022 10:11 PM

This extension allows animating components and widgets (RecyclerView Widgets).

Methods

AnimateRecyclerWidget

This method applies an animation on a widget of the RecyclerView. Duration and delay are in milliseconds.
