Background

It is supported for:KodularAppInventorNiotron
It is part of Fancy RecyclerView Widgets
To get this extension, please go to: Fancy RecyclerView Widgets

Posted on January 25, 2022 10:10 PM

Creates a background for widgets. The properties are used to create the style of the widgets. The methods are used when the widgets have already been created.

Methods

Returns a new property set containing the previously defined properties.

Applies a color to the background. Info:

Unlike the SetColor method, this one will apply a color over the entire background, including the edges.

The text should be a path to a file or the name of a file found in the assets of the project. If this block is used, all other properties will be ignored to only display the image. Supports NinePatch images.
